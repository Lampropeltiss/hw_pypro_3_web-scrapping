## Веб-скрапинг: программа для парсинга статей

Реализована программа для парсинга статей с [заданной страницы](https://habr.com/ru/articles/) и выбора тех статей, в которых встречается хотя бы одно из ключевых слов. 

Использованы **BeautifulSoup** и **регулярные выражения**.

Ключевые слова определяются в начале скрипта. Поиск ведётся по всей доступной preview-информации, т. е. по информации, доступной с текущей страницы. Список подходящих статей выводится в консоль в формате: <дата> – <заголовок> – <ссылка>.

Также реализовано улучшение скрипта таким образом, чтобы он анализировал не только preview-информацию статьи, но и весь текст статьи целиком. Для этого реализовано получение страниц статей и поиск по тексту внутри этих страниц.
